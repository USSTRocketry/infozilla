Detailed Description

A class to interface with the RYLR998 LoRa radio module.

This class provides methods to initialize and communicate with the RYLR998 LoRa radio module. It allows sending and receiving data, setting frequency, transmission power, and configuring LoRa parameters such as spreading factor, bandwidth, and coding rate.

Note
This class inherits from the TelemetryRadio base class.
Parameters
rylr998Reference to a hardware serial port used for communication with the RYLR998 module.
baudRateThe baud rate for serial communication. Default is 9600.
