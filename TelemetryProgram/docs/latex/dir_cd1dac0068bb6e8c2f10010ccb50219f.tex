Header-\/only library for conversion to/from half-\/precision floating point formats\hypertarget{README.md_autotoc_md1}{}\doxysubsection{\texorpdfstring{Features}{Features}}\label{README.md_autotoc_md1}

\begin{DoxyItemize}
\item Supports IEEE and ARM alternative half-\/precision floating-\/point format
\begin{DoxyItemize}
\item Property converts infinities and Na\+Ns
\item Properly converts denormal numbers, even on systems without denormal support
\end{DoxyItemize}
\item Header-\/only library, no installation or build required
\item Compatible with C99 and C++11
\item Fully covered with unit tests and microbenchmarks
